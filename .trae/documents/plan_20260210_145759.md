# Dashboard Redesign Plan for Responsive "Clean" Layout

## Problem Analysis
The "Recent Classifications" table on the dashboard is truncated on smaller screens (laptops), causing horizontal scrolling or hidden columns. The current layout is too dense for smaller viewports, and the user wants a "cleaner" look that shows only necessary info.

## Proposed Solution: Responsive Card List Pattern
Instead of forcing a full table on smaller screens, we will switch to a **Responsive List Layout**.
- **Large Screens (XL)**: Keep the full table with all details (Product, Code, Dossier Status, Actions).
- **Laptops/Tablets (<XL)**: Switch to a "Card List" view (similar to the mobile view but optimized for density). This hides less critical columns (like raw Code digits if space is tight) and stacks information vertically to save horizontal space.

## Design Changes
1.  **Hide Table on Laptops**: Change the table visibility from `hidden md:block` to `hidden xl:block`.
2.  **Show List on Laptops**: Enable the "Mobile/Tablet" card layout for screens up to `xl` (`md:hidden` -> `xl:hidden`).
3.  **Optimize Card Content**:
    -   **Product Name**: Primary focus, truncated if too long.
    -   **Code**: Show formatted CN code below the name.
    -   **Status**: Use a small badge/icon for Dossier status instead of a full column.
    -   **Action**: Single "View" button + Delete icon, compact layout.

## Implementation Steps
1.  **Modify `src/app/(app)/dashboard/page.tsx`**:
    -   Update the CSS classes for the "Card Layout" section to be visible up to `xl` breakpoint.
    -   Update the CSS classes for the "Table Layout" section to be visible only from `xl` upwards.
    -   Refine the existing card layout to be "cleaner" (less padding, better alignment) so it looks good on laptops, not just phones.

## Outcome
-   **Cleaner Look**: No horizontal scrolling or cramped columns on laptops.
-   **Impressive UI**: Modern "card" aesthetic for smaller screens, professional data table for large monitors.
-   **Responsiveness**: seamless adaptation from MacBook Air (13") to large Desktop monitors.
